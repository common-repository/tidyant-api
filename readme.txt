Create a repair into your TidyAnt web from your wordpress site.

== Description ==

TidyAnt API is a plugin that creates a form in your website to send repairs to your TidyAnt web. You need a [TidyAnt API](https://tidyant.com/) web for using this plugin.

== Installation ==

1. Login to your WordPress admin.
2. Click on the plugins tab.
3. Click the Add New button.
4. Search for "TidyAnt API".
5. Look for our plugin, click "Install Now" and then "Activate".
6. Once the plugin is installed and active, you need to configure your credentials for connect with your TidyAnt site.
7. Last step is add the wildcard {tidyant} in the page that you will use for show the new repair form.

== Frequently Asked Questions == 
= How I can show the repair form?? =
You need to add the wildcard {tidyant} into a post or a page. The wildcard will be replaced by the repair form.
